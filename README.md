# tweezel

TweezeL Is A Free Tools For Creating Twine Games On Android.  
This Tools Build With **VueJS 3 + Tailwind Css** For Styling.

Link Tools : https://tweezel.netlify.app

For Now It Only For `Sugarcube v2` Story Formats. Will Support Other Formats In The Future.  
(Story Formats That Listed On The Twine Apps).

I Think This Tools Already Usable. :3

**If U Are Already Tried Build Games On Twine Or Tweego Using Sugarcube v2.  
I Want U To Try Mine And Tell Me What Should I Add The Feature First.**

<hr>

For Now, You Can :
- [x] Add, Edit and Delete Story.
  - [x] Edit Title Story.
  - [x] Generate IFID. (Now Generate IFID Only When Creating Story.)
  - [x] Set Story Start
  - [x] Create Passage.
  - [x] Delete Passage.
  - [x] Edit Passage.
- [x] Use Custom Javascript And Css.
- [x] You Can Pretty Much Use All Sugarcube `< v2.36.1` Syntax.
- [x] Run Edited Story.
- [x] Backup And Restore All Story Or Some Of Them.
- [x] Can Export To .html File.
- [x] Built In Tutorial.
- [x] Also Dark Theme. Light Attract Bugs. :3
- [x] Finally... You Can Re-Order The List.
